# Shoulder_Pi
This is a general purpose game engine that runs on the Raspberry Pi 3 using ARM assembly.

## Important notes
You need to be in console mode (shortcut Ctrl+Alt+F1) to run the code
To get back to GUI mode, press Ctrl+Alt+F7
